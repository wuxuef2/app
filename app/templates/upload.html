Upload

Drag and drop made easy

Instantly add drag and drop for file uploads with simple file input. 

<input type="file" name="file" data-tools="upload" data-url="/upload.php">

In a file handler output JSON with a link to a file, for example, for PHP: 

echo json_encode(array('file' => 'http://site.com/files/my-file'));
